POBIERANIE

Wystarczy kliknąć na zielony przycisk Code i kliknąć na przycisk Download ZIP. Plik zostanie pobrany w formacie zip. Należy go wypakować w dowolnym miejscu.

URUCHAMIANIE

1. Zainstaluj środowisko programistyczne języka Python (min. 3.11.5). Upewnij się, żeby dodać Pythona do zmiennych środowiskowych.
2. Uruchom dowolny terminal. Przenieś się w nim do ścieżki, w której został wypakowany plik zip.
3. Wpisz w terminalu: 'pip install -r ./requirements.txt'. Zainstaluje to wszelkie zależności projektu.
4. Wpisz w terminalu: 'py ./pasjans.py'. Uruchomi to grę.

UWAGA!

Zmienianie rozmiaru okna/znaków terminalu, w którym ma miejsce rozgrywka, jest niewskazane.

STEROWANIE

Menu

Strzałki - poruszanie wskaźnikiem
Enter - dokonanie wyboru
Escape - wyjście z gry

Rozgrywka

Strzałki - poruszanie wskaźnikiem
Enter - wybranie karty/miejsca jej przeniesienia
Backspace - dobór karty ze stosu rezerwowego
U - cofnięcie ruchu
Escape - pauza
